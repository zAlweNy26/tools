# isOfType()

> **isOfType**\<`T`\>(`value`): `value is T`

Type guard function that checks if a value is of a certain type.

## Type Parameters

| Type Parameter | Description |
| ------ | ------ |
| `T` | The type to check for. |

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `value` | `unknown` | The value to check the type of. |

## Returns

`value is T`

A boolean indicating whether the value is of the specified type.
