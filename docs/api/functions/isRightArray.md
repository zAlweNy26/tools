# isRightArray()

> **isRightArray**\<`T`\>(`array`): `array is T[]`

Type guard function that checks if an array is of type T[].

## Type Parameters

| Type Parameter | Description |
| ------ | ------ |
| `T` | The type of the elements to check for. |

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `array` | `unknown`[] | The array to check. |

## Returns

`array is T[]`

A boolean indicating whether the array is of the specified type.
