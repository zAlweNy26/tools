# getLCP()

> **getLCP**(`str1`, `str2`): `number`

Returns the length of the longest common prefix between two strings.

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `str1` | `string` | The first string to compare. |
| `str2` | `string` | The second string to compare. |

## Returns

`number`

The length of the longest common prefix between the two strings.
